Planets

I want to 3d print each planet as a half sphere that attaches to a stand. I want to use this tactic so they can feel the differences in the sizes of the planets. Because they have never seen anything, it must be hard for them to understand the size differences, and how small our planet is in comparison to all the others. I have 8 half spheres that represent each planet, and each has a stand. They will all be detachable so they can easily hold and compare. Each different stand is numbered, and I will be including a voice description saying which planet is which by going in order. Also, each stand will clip into one another. Along with the terrestrial planets (the ones with a rocky outer layer surrounded by a shallow atmosphere) will be bumpy, and the gaseous planets will be smooth.

You can tell which stand is which by using the braille numbering system labeled on it. If you grab the first stand (feel around for the numbers) then we can begin. Hold the stand by having the braille marker as the right corner facing closest to you. The first smallest planet is Mercury. Then if you go one to the left, that is Mars. Then if you keep following the order, it goes Venus, then Earth, then Neptune, then Uranus. If you look at the second stand with the braille for 2. This stand has Saturn on it. Finally, if you grab the third stand, then that is the stand for Jupiter. Take it out and feel the size! Feel how small Earth is in comparison. Crazy right!
